1. github上创建工程
2. 文件夹内打开git bash
3. 输入git init
4. 输入git clone "/要复制的工程/"
5. 输入git status查看文件
6. git add "/文件名/"
7. git commit -m “/"相关备注"/”(需要双引号)
8. git push -u origin master
搞定！

出错1：fatal: 'origin' does not appear to be a git repository
解决办法：git remote add origin "/要复制的工程/"

错误2：error: failed to push some refs to 'git@github.com:DingChujiang/Python-.git'
解决办法：git pull --rebase origin master

出错3：文件大小超过100M
解决办法：git lfs install(只需安装一次) git lfs track “/"文件"/"(需要双引号)，然后执行6-8即可
